Memuat Extension
---

1. Buka ```chrome://extensions``` di browser Chrome.
2. Aktifkan Developer Mode.
3. Klik Load unpacked dan pilih folder yang sudah diclone.
4. Ikon extension Anda akan muncul di toolbar browser.
